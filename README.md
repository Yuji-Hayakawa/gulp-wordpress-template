# gulp-wordpress-template

## 環境情報
1. Node `v18.12.1`
1. WordPress `v.6.1`
## 初回インストール手順

1. リポジトリをクローン
   ```zsh
   git clone https://github.com/Yuji-Hayakawa/gulp-wordpress-template.git
   ```
1. Node パッケージのインストール（必要あれば）
   ```zsh
   npm install
   ```

## 実行方法

### SCSSをコンパイル&ローカルサーバー起動＆ライブリロード etc...

```zsh
npx gulp
```
詳細は gulpfile.js を確認してください。
